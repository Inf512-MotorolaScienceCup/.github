# Inf512

### Historia i skład

Nasza drużyna postawiła sobie jedno zadanie, a mianowicie wykorzystać posiadane umiejętności jak najlepiej, nauczyć się jak najwięcej i osiągnąć jak najwyższe miejsce w konkursie. Poniżej znajdują się odnośniki do repozytoriów z pracami konkursowymi. Razem uzgodniliśmy, że wykonamy `Gre PANG` oraz `Symulator układów logicznych`, wydały się nam one najciekawsze i najambitniejsze spośród podanych tematów.

W skład naszego zespołu wchodzą: 
 - Oliwier Angielski
 - Antoni Bizoń
 - Bartosz Kozieł
 - Paweł Pasternak
 - Szymon Sadowski
